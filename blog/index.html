Stocks with Excel support

Tested Environment

Ubuntu 16.04, 12.04. LibreOffice calc

Purpose

In Internet there are lot of tools to view current stock price, its bid quantity. But there are very few website supports real time portfolio support, where the stocks can be added or sold to analyse the personal capability about correct entry level and exit level from market to book reasonable profit. If you are looking for custom portfolio management with real time stock value update in excel then this post is for you

How it Works

With the help of excel sheet managing custom portfolio, Current profit based on categorised portfolio will be possible.
